# Demo: Vulnerable Bot (CTF target intencional)

Chatbot intencionalmente vulnerable a prompt injection, jailbreaks y PII leakage.
Diseñado para practicar red teaming con Garak, PyRIT y DeepTeam.

⚠️ **SOLO para uso local en entornos de lab.** No exponer en producción.

**Stack:** FastAPI + LLM local (Ollama)

**Levanta con:** `docker compose -f docker/compose.yml up vulnerable-bot`

**Usado por:** módulos 07, 08, 09
