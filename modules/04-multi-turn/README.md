# Módulo 04 — Multi-Turn Testing

**Status:** planned

## Objetivos

- Escribir tests de conversaciones multi-turno usando `ConversationalTestCase` de DeepEval
- Medir `KnowledgeRetention` y `ConversationCompleteness` en flujos conversacionales
- Diseñar suites de testing que cubran flujos completos de usuario extremo a extremo

## Capítulo del manual

Este módulo cubre los conceptos de la sección X del manual (`docs/02-tipos-de-testing.md`).

## Conceptos clave

- ConversationalTestCase en DeepEval para testing de diálogos
- KnowledgeRetention: ¿recuerda el chatbot información de turnos anteriores?
- ConversationCompleteness: ¿se resuelven los objetivos del usuario?
- Estrategias de testing para flujos multi-turno con estados complejos
